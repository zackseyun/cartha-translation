# sources/nag_hammadi

Phase E source registry for Nag Hammadi translation prep.

## Texts in scope

- Gospel of Thomas
- Gospel of Truth
- Thunder, Perfect Mind

## Directory notes

- `texts/*/manifest.json` — source-of-truth config for each text
- `segment_index/` — generated segment maps for saying / section / line-block work
- `raw/` — tracked HTML/text/JSON witness snapshots from online sources
- `staging/` — local PDFs/images waiting to be registered for OCR
- `ocr/jobs/` — tracked OCR job stubs
- `ocr/output/` — generated OCR output landing zone
- `witnesses/` — built witness bundles per text
- `prompts/` — generated translator prompts per text
